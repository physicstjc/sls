AI Generated Apps for Student Learning Space

Random Name Generator

For use in class. Either enter the names with semi-colons as separators or upload a classlist (consisting only of names, not index number, etc) in csv form.

Test it out Download for SLS

Prompts

Provide the code for the following in a single html file:

  1. This web page lets you generate random names from a list of names you provide.
  2. When you visit the page, you'll see a title "Random Name Generator" at the top.
  3. Below the title, there's a section labeled "Enter names separated by semicolons."
  4. You can type or paste a list of names into a box provided. Each name should be separated by a semicolon.
  5. Alternatively, you can upload a file containing a list of names in CSV format using the "Or upload a CSV file" option.
  6. Once you've entered the names or uploaded the file, click the "Submit" button.
  7. After submitting, the page will display another section.
  8. In this new section, there's a button labeled "Generate Random Name."
  9. When you click this button, a random name from the list you provided will appear on the page.
  10. Below the "Generate Random Name" button, there's a paragraph displaying the randomly generated name.
  11. If you want to remove the displayed name from the list, you can click the "Remove Name" button below the displayed name.
  12. Any names that you've generated will be listed below the "Generated Names" section.
  13. Each generated name will appear on a separate line.
